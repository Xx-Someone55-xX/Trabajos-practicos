Explicación del Producto Mixto

1. Definición Formal

El producto mixto (o triple producto escalar) de tres vectores en ℝ³ combina el producto vectorial y escalar:

a · (b × c)

2. Interpretación Geométrica

El valor absoluto del producto mixto representa el volumen del paralelepípedo definido por los tres vectores:

  • Positivo: Orientación derecha (regla de la mano derecha)
  • Cero: Vectores coplanares (linealmente dependientes)
  • Negativo: Orientación izquierda

3. Propiedades Clave

Permutación Cíclica

a·(b×c) = b·(c×a) = c·(a×b)

El producto mixto mantiene su valor bajo permutaciones cíclicas de los vectores. Esto significa que podemos rotar el orden de los vectores sin cambiar el resultado. Sin embargo, cambiar el orden no cíclico altera el signo del resultado.

Anticonmutatividad

a·(b×c) = -a·(c×b)

El producto mixto cambia de signo cuando se intercambian dos vectores. Esta propiedad proviene de la anticonmutatividad del producto vectorial. Es importante para determinar la orientación del sistema de vectores.

Linealidad

Lineal en cada argumento

El producto mixto es lineal en cada uno de sus argumentos. Esto significa que cumple con las propiedades de aditividad y homogeneidad para cada vector:

(αa + βa')·(b×c) = αa·(b×c) + βa'·(b×c)

Esta propiedad permite descomponer cálculos complejos en partes más simples.

Condición de coplanaridad

a·(b×c) = 0 ⇔ a, b, c son coplanares

Cuando el producto mixto es cero, indica que los tres vectores están en el mismo plano (son linealmente dependientes). Esta propiedad es fundamental para verificar la independencia lineal de vectores en ℝ³.

4. Cálculo mediante Determinante

Se puede calcular como el determinante de la matriz formada por los vectores:

det(a, b, c) = |a₁ a₂ a₃| |b₁ b₂ b₃| |c₁ c₂ c₃|

Que se desarrolla como:

a₁(b₂c₃ - b₃c₂) - a₂(b₁c₃ - b₃c₁) + a₃(b₁c₂ - b₂c₁)

5. Aplicaciones Prácticas

  • Cálculo de volúmenes en geometría
  • Determinar si vectores son coplanares
  • Física: torque, momento angular
  • Computación gráfica: normales a superficies
